PHP Data Structures Extension

Introduction

The PHP Data Structures Extension for PHP 7 provides the solutions and improved performance of classical data structures and is a nice alternative to PHP's one-fize-fits-all array. The classes—of Vector, Deque, Map, Set, Stack, Queue, PriorityQueue and Pair—and their interfaces—Collection, Hashable and Sequence—are explained in Efficient Data Structures for PHP written by extension's author. They are also discussed in this Reddit thread.

Answers to questions can be found at the end of the Ds github repository README.

There is also another—I believe—related extension, polyfill.

The Collection Interface

The Collection interface covers functionality common to all the data structures in this library. It guarantees that all structures are traversable, countable, and can be converted to json using json_encode(), and it thereby
provides support for foreach, echo, count, print_r, var_dump, serialize, json_encode, and clone.

Importanat Comment on Installation
